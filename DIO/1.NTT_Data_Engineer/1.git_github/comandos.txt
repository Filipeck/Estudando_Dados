git config --global
git config --global user.name (passar nome entre "")
git config --global user.email
git config init.defaultBranch (configura o nome padrão da branch)
git config --global --list (mostra todas as configurações que já fizemos no global)
git config credential.helper store (outra opção é cache, quando usado store a credencial ficará armazenada, já a cache é caso a máquin seja compartilhada)
git clone "seu local clonado" nome-pasta-local(insere o local que será clonado)
git remote -v (mostra o repositório remoto que está vinculado)
git remote add origin URL (adiciona um repositório local a um remoto)
